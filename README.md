## 위치 권한이 거부되어 서울로 표시될 때 (해결 방법)

브라우저에서 위치 권한을 거부하면, 앱은 현재 위치를 가져올 수 없어 **기본값(서울)** 으로 날씨를 조회합니다.

### 해결 방법 (Chrome 기준)

1. 브라우저 주소창 왼쪽의 **🔒 자물쇠 아이콘** 클릭  
2. **사이트 설정**(또는 권한) 메뉴로 이동  
3. **위치(Location)** 권한을 **허용**으로 변경  
4. 페이지 **새로고침(F5)**

위 설정 후에는 "서울" 대신 **현재 위치** 기준으로 날씨가 표시됩니다.
