## 📖 프로젝트 소개

**"오늘 뭐 입지?"** 는 사용자의 현재 위치를 기반으로 실시간 날씨 데이터를 가져와, 기온·바람·습도 등을 종합적으로 분석한 **체감온도**에 맞는 의상을 추천해주는 웹 애플리케이션입니다.

단순히 기온만 보는 것이 아니라, **체질(추위/더위 민감도)**, **성별**, **스타일 선호도**까지 고려하여 개인 맞춤형 추천을 제공합니다.

---

## ✨ 주요 기능

### 🌡️ 실시간 날씨 연동
- **기상청 초단기실황 API** 연동으로 실시간 기온, 습도, 풍속, 강수 정보 제공
- **브라우저 Geolocation API**를 활용한 현재 위치 자동 감지
- 위도/경도 → 기상청 격자 좌표 자동 변환

### 👕 스마트 의상 추천
- 체감온도 기반 아우터 / 상의 / 하의 카테고리별 추천
- **날씨별 분류**: 4단계 날씨 카테고리별 최대 2개씩 표시
  - 추운 날씨 (-100°C ~ 5°C)
  - 쌀쌀한 날씨 (6°C ~ 15°C)
  - 따뜻한 날씨 (16°C ~ 22°C)
  - 더운 날씨 (23°C ~ 100°C)
- **성별** 선택 (남성 / 여성 / 공용)
- **스타일** 선택 (캐주얼 / 모던 / 스트리트 / 공용)
- **체질** 선택 (보통 / 추위 잘 탐 / 더위 잘 탐)
- **카테고리 필터**: 아우터/상의/하의/전체 선택 가능

### 🗄️ 내 옷장 관리
- **커스텀 의상 추가**: 이미지 파일 업로드 (1MB 이하)
- **23개 기본 의상**: 다양한 카테고리의 기본 의상 데이터
- **커스텀 의상 제거**: "커스텀 의상 모두 제거" 버튼으로 일괄 정리
- **드래그 앤 드롭**: 의상을 아바타에 직접 장착
- `localStorage`를 활용한 데이터 영구 저장

### 👤 아바타 입어보기
- 추천 아이템 또는 내 옷장 아이템을 아바타에 레이어링
- 아우터 / 상의 / 하의 레이어 분리 렌더링

### 🌙 다크 모드
- 원클릭 다크/라이트 모드 전환
- CSS 변수 기반 테마 시스템
- 설정값 `localStorage` 자동 저장

### 💬 상황별 어드바이스
- 어제 대비 기온 비교 안내
- 일교차, 비, 미세먼지 등 상황에 따른 맞춤 조언

## � 사용 방법

### 1. 기본 설정
1. **성별 선택**: 남성/여성/공용 중 선택
2. **스타일 선택**: 캐주얼/모던/스트리트/공용 중 선택
3. **체질 선택**: 보통/추위 잘 탐/더위 잘 탐 중 선택
4. **위치 권한 허용**: 브라우저가 위치 정보 접근을 요청하면 허용

### 2. 의상 추천 확인
- 현재 날씨에 맞는 의상이 **날씨 카테고리별**로 표시됩니다
- 각 카테고리별로 **최대 2개**의 의상이 추천됩니다
- 날씨 조언과 추가 팁을 확인합니다

### 3. 의상 조합
- 추천된 의상을 **드래그**하여 오른쪽 코디네이터에 놓습니다
- 카테고리에 맞는 의상만 장착할 수 있습니다
- X 버튼으로 장착된 의상을 제거할 수 있습니다

### 4. 커스텀 의상 관리
- **"의상 추가"** 버튼으로 개인 의상을 추가합니다
- 이미지 파일을 업로드하여 시각적으로 표시할 수 있습니다 (1MB 이하)
- **"커스텀 의상 모두 제거"**로 추가한 의상을 정리할 수 있습니다

### 5. 카테고리 필터링
- **전체**: 모든 카테고리의 의상 표시
- **아우터**: 아우터 카테고리만 표시
- **상의**: 상의 카테고리만 표시
- **하의**: 하의 카테고리만 표시

## �🚀 시작하기

### 1. 클론

```bash
git clone https://github.com/SIMHEONSEOB/cloth_recommand.git
cd cloth_recommand
```

### 2. 실행

별도의 빌드 과정 없이 `index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

> **Live Server 사용 권장**: VS Code의 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 확장을 사용하면 ES Module 지원과 실시간 리로드가 가능합니다.

### 3. 위치 권한 허용

앱 실행 시 브라우저가 위치 권한을 요청합니다. **허용**을 선택하면 현재 위치 기반 날씨를 제공합니다.

> ⚠️ 위치 권한을 거부하면 **서울** 기준으로 날씨가 표시됩니다.

#### 위치 권한 재설정 방법 (Chrome)

1. 주소창 왼쪽의 🔒 **자물쇠 아이콘** 클릭
2. **사이트 설정** → **위치(Location)** 권한을 **허용**으로 변경
3. 페이지 **새로고침 (F5)**

## 🔧 체감온도 계산 공식

본 앱은 **Wind Chill Index** 기반의 체감온도 공식을 사용하며, 체질에 따라 보정값을 적용합니다:

```
체감온도 = 13.12 + 0.6215×T - 11.37×V^0.16 + 0.3965×T×V^0.16
```

- `T` = 현재 기온 (℃)
- `V` = 풍속 (m/s)
- 추위 민감: -2℃ 보정  
- 더위 민감: +2℃ 보정
